ForgottenQuests
ForgottenQuests добавляет настраиваемую систему квестов с наградами, доставкой предметов, охотой на существ, заданиями на координаты, таймерами отката и встроенным редактором квестов прямо в игре.

Разработано для сервера Forgotten Legends.

 
Возможности
Типы заданий
Collect Item - Добыть предмет
Submit Item - Отдать предмет
Kill Entity - Убийство существа
Reach Position - Дойти до координат
Custon text - Можно использовать для стартовых вещей или других задумок по желанию, не требует выполнения каких либо действий
 
Встроенный редактор квестов
Сетка наград 3x3
Настройка через drag & drop
Поддержка предметов и блоков
Награды автоматически:
помещаются в инвентарь игрока
выбрасываются рядом с игроком при отсутствии места
 
Встроенный редактор квестов
Игроки с Creative Mode или привилегией QuestGiver могут:

Создавать квесты
Редактировать квесты
Настраивать награды
Настраивать таймеры отката
Настраивать цели заданий прямо в игре
Редактирование JSON вручную не требуется.

 
Редактор отката
Поддерживаются реальные таймеры:

почасовые
ежедневные
повторяемые квесты
Откат использует реальное время, а не игровое.

 
Убийства существ
Поддерживаются:

точные цели существ
отслеживание прогресса
уведомления о прогрессе в чате
 
Поиск координат
Игрок завершает квест, когда входит в указанную область координат.

 
Горячая клавиша
Кнопка открытия меню по умолчанию - V

Кнопку можно изменить в:

Настройки -> Управление -> ForgottenQuests

 
Уведомления в чате
Когда цель задания выполнена, игрок получает цветное сообщение в чат.

Пример:

Квест "Hunter" выполнен! Нажмите [V] чтобы открыть меню заданий.

Отображаемая клавиша автоматически обновляется, если игрок переназначил hotkey.

 
Привилегии
ForgottenQuests добавляет пользовательскую привилегию QuestGiver

Игроки с этой привилегией могут создавать и редактировать квесты без Creative Mode.

Выдать привилегию:

/player [playername] privilege grant QuestGiver

 
Конфигурационные файлы
